### Controls

**[a]/[q]** add a vertex

**[g]** grab a vertex

**[d]** delete a vertex

**[b]** toggle fuzzy balls

**[v]** add random vertices

**[s]** toggle simplices

**[i]** display info

**[c]** (k+1)-NN

**[x]** (k-1)-NN
